Documentation

{{ site.name }} leverages Bootstrap classes which means if you have a customized version of Bootstrap, {{ site.name }} will automatically take on your customizations.

Dependencies

  1. jQuery
  2. Bootstrap
  3. Moment.js
Make sure to include required stylesheets and scripts like so (making sure paths point to your actual files): 
<link href="bootstrap.min.css" rel="stylesheet">
<link href="bootstrap-datetime.min.css" rel="stylesheet">

<script src="jquery.min.js"></script>
<script src="bootstrap.min.js"></script>
<script src="moment.min.js"></script>
<script src="bootstrap-datetime.min.js"></script>

There are a few ways to get started with {{ site.name }}:

  1. Automatically apply {{ site.name }} to all datetime, date, and time inputs with the following JavaScript: 

    bootstrapDateTime.auto();
    Note: The inputs must exist in the DOM before calling this line of code. To be safe you may want to add it to jQuery document ready like so: 
    jQuery(function () {
  bootstrapDateTime.auto();
});

  2. Apply {{ site.name }} to only certain DOM elements using a jQuery selector (In this example apply {{ site.name }} to all DOM elements with the class enhancify): 

    bootstrapDateTime.bind('.enhancify');

  3. Apply {{ site.name }} to a jQuery object (In this example apply {{ site.name }} to all DOM elements with the class enhancify). 

    var elements = jQuery('.enhancify');
bootstrapDateTime.bind(elements);

  4. Apply {{ site.name }} to a DOM element (In this example apply {{ site.name }} to DOM element with the id enhancify). 

    var element = document.getElementById('enhancify');
bootstrapDateTime.bind(element);
