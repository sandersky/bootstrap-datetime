How to Build

To perform static code analysis and compile the source code, {{ site.name }} uses Grunt. To manage third-party asset libraries such as jQuery, {{ site.name }} uses bower. If you do not already have Node and npm installed, that will be your first step which is not documented here. Once you have installed both Node and npm you can following the remaining installation instructions below.

Install Grunt

Install grunt-cli globally using npm:
npm install -g grunt-cli

Install Bower

Install bower globally using npm:
npm install -g bower

Install packages and dependencies:
bower install

Grunt Commands

Analyze and Build

If you want to analyze the code (static code analysis) and compile it, run the following command:
grunt

Analyze

If you don't care about compiling and just want to perform static code analysis run the following command:
grunt test

Build

To just compile the code and skip the testing step run the following command:
grunt build

Build CSS

To just compile the CSS run the following command:
grunt build-css

Build JavaScript

To just compile the JavaScript run the following command:
grunt build-js
